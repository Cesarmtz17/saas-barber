# SaaS Barber - Sistema de Citas

Plataforma **multitenant** tipo Calendly para barberías. Permite a negocios gestionar citas, servicios y horarios desde un dashboard.

## Características
- Sistema multitenant (múltiples negocios)
- Gestión de citas y servicios
- Dashboard para propietarios
- Vista pública para clientes

## Tecnologías
- Django 5.0
- Tailwind CSS
- SQLite/PostgreSQL

## Instalar
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
